# Tir38 Tap

## How do I install these formulae?

`brew install tir38/tap/<formula>`

Or `brew tap tir38/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "tir38/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
